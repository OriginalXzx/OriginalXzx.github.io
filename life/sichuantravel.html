      国庆四川出游计划

      originalxzx 2019-08-09 travel

      达瓦更扎-阿坝

      本次出游备选地点：达瓦更扎、色达、阿坝!

      出行第一步，先看路线及耗时，本次出游选择自驾游。

      成都-达瓦更扎  驾车4小时，过路费53元

      达瓦更扎-阿坝 驾车6小时

      阿坝-色达 驾车7小时

      考虑到时间原因，所以本次旅行两个主要地点达瓦更扎 、阿坝

      # 达瓦更扎

      达瓦更扎在藏语中意为“美丽的神山”，景区主要因云海、日出、佛光、雾凇、草甸、高山杜鹃等景观而闻名。


      单凭这两张图就值得我们去看了，再拍点照片，岂不美哉！ 再看看大体时间上安排： ### 成都—宝兴—达瓦更扎 * 早上：08:00 成都出发，上成雅高速到达雅安市，再经飞仙关---芦山县---宝兴县--- 跷碛乡（车程 5 小时） * 中午：11:30 在嘎日村享用午餐。 * 下午：13:30 到达跷碛湖，停车拍照留念，各种装逼合照，先在朋友圈打个卡，随后大约 14:30 到达神木垒景区游玩到晚上，参加藏家篝火晚会（这里为什么不直接上海搭帐篷呢，考虑到没有露营的经验，如果后续有必要也不是不可以直接山上搭帐篷）。 * 第二天凌晨四点，乘当地车上瓦更扎看雾凇奇峰，这里天空安详，碧空如洗，不染尘世。更能让我们放空一切与大自然对话，甚至不顾一切的高声呐喊，感悟在人世间最渺小的存在。蓝天白云，还有放养的猪，在海天一色的世界尽情享受这无边无际的世外桃源。看完日出云海后下山。7-9点看日出云海，九点左右坐车下山，预计10点到达山下，继续下一段旅程。

      【装备提示】

      1.登山鞋或者运动鞋，冲锋衣或薄款羽绒服等。

      2.水壶水袋，垃圾袋，毛巾（湿巾）。

      3.帽子、墨镜纸等。

      4.身份证，足够现金。

      # 阿坝

      本次阿坝之旅主要为了感受那种无雕刻的自然风光，以及当地的藏族风情,感受不一样的诗与远方。

      # 桃坪羌寨-甘堡藏寨-毕棚沟

      阿坝县是内地通往 西北 的重要交通枢纽，是重要的商贸中心和物资集散地,所以小物品一定比较多，可以挑选作为纪念，当然也可以感受当地的文化，风土。

      # 桃坪羌寨

      桃坪羌寨是羌族建筑群落的典型代表，电影《杀生》曾在这里取景。石头与黄泥砌成的建筑经历了无数的地震后仍完好无损，被称为保存最完整的羌族建筑文化艺术“活化石”。

      # 甘堡藏寨


      甘堡藏寨有着2000多年的历史，如石头城堡般的寨子至今仍保存完好。整座寨子背负青山，面临江水，耸立在大雪山的脚下，建筑错落有致，景色非常美。在这里可以体验藏族风情。

      # 毕棚沟


      10月份过去，恰好能欣赏毕棚沟的秋景。

      【时间安排】 10-2 达到达瓦古扎 10-03 下午到阿坝县停留一晚 10-04 桃坪羌寨 + 甘堡藏寨，晚上到毕棚沟附近休息

      # 色达


      色达 在藏语中意为“金马”,在这里，你或许能看到远离烟火的生命本来的样子，去享受生命的本身，去聆听生命的本身，也看看生命的终结。

      # 天葬 + 佛学院


      天葬是藏区的最普遍一种丧葬习俗。按当地习俗，人死后需停尸数日，请喇嘛念经择日送葬。天葬师随即将尸体衣服剥去，按一定程序肢解尸体，肉骨剥离。最后用哨声呼来鹫，按骨、肉顺序别喂食。 天葬台在佛学院至 色达 县城之间，每天天葬的时间比较固定，14:30分——15：00分左右。

      【时间安排】 10-5号 直接从毕棚沟出发开车到色达，到了色达自行安排。 10-6号 返程回成都。
